1. 버프가 끝나기 전에 같은 버프가 주어졌을 때, 이전 버프가 끝나도 새로 갱신된 버프를 삭제하지 않는 방법.

2. 만약 Always Type의 Buff를 가진 캐릭터의 stat이 업데이트 되었을 때, 기존에 Buff를 받고 있던 캐릭터들의 Buff를 갱신하는 방법

Red : 공격력, 공격 속도
Green : 공격 범위, 공격 대상, 이동 속도
Blue : 스킬 쿨타임, 스킬 확률

ㅇ Deal
xx : 궁수
- 치명타 스킬 + 개인 스턴(확률)
RRx : 레인저
- 치명타 스킬 + 개인 스턴 (확률)
- 개인 공속 증가 스킬 + 고정 데미지 증가 (쿨타임)
GGx : 해적 선장
- 범위 데미지 스킬 + 범위 방깎(확률)
- 범위 공증 + 범위 공속 증가(쿨타임)
BBx : 대마법사
- 범위 데미지 스킬 + 범위 스턴 (쿨타임)
- 범위 이속 감소 (항상)
RGB : 영웅
- 치명타 스킬 + 개인 스턴 (확률)
- 범위 공증 + 범위 공속 증가(항상)

ㅇ Tank
xx : 전사
- 범위 데미지 + 범위 스턴 (쿨타임)
RRx : 바이킹
- 치명타 스킬 + 개인 스턴 (확률)
- 개인 공속 증가 + 가장 센 유닛 공속 증가 (쿨타임)
GGx : 기사
- 범위 데미지 + 범위 스턴 (확률)
- 범위 데미지 + 범위 스턴 (쿨타임)
BBx : 장군
- 범위 데미지 + 범위 스턴 (확률)
- 범위 공증 + 범위 공속 증가 (쿨타임)
RGB : 왕
- 범위 공증 + 범위 이속 증가 (항상)
- 범위 데미지 + 범위 스턴 (쿨타임)

ㅇ Heal 
xx : 집시
- 가장 센 유닛 공증 (쿨타임)
RRx : 드루이드
- 가장 센 유닛 공증 + 고정 데미지 증가 (쿨타임)
- 범위 데미지 + 범위 이속 감소 (확률)
GGx : 공주
- 범위 공증 + 범위 공속 증가 (쿨타임)
- 범위 방깎 + 범위 이속 증가 (항상)
BBx : 마녀
- 범위 데미지 + 범위 스턴 (쿨타임)
- 범위 이속 감소 (항상)
RGB : 여왕
- 가장 센 유닛 공증 + 고정 데미지 증가 (쿨타임)
- 범위 공증 + 범위 공속 증가 (항상)
